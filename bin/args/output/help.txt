
Matic.js
---------

  Arguments
  ---------

    help
    ----

    help, -help, --help, -h, --h:

      Matic help.

    version
    -------

    version, -version, --version, -v, --v:

      Display Matic.js version number.

  Usage
  -----

  Simply type 'matic', for more detailed information about how to set
  up a project folder for building please visit the github examples at
  https://github.com/mattyod/
